# Tutorial

Abrir a pasta /Locacao e rodar o seguinte comando: 
```
 mvn spring-boot:run
```
Com isso, terá o administrador:
```
login: username

senha: senha
```

E uma locadora com uma locação. O restante, poderá ser feito pela aplicação Web e pelo Postman.
